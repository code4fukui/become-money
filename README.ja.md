# become-money


徳島県上勝町のオープンデータを使用し、品目がお金になるかどうかをテストするクイズゲームです。

## デモ
https://code4fukui.github.io/become-money/

## 特徴
- 品目がお金になるかどうかを問う2択クイズゲーム
- 徳島県上勝町のリサイクル分別オープンデータを使用
- 各問題の後に正解とリサイクルの詳細を表示
- プレイヤーの最終スコアと正解率を表示

## データ / API
クイズのデータは、[上勝町リサイクル分別オープンデータ](https://github.com/code4fukui/zerowaste/blob/main/README.md)プロジェクトから取得しています。
