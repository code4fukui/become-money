# become-money

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A quiz game that tests whether items can become money, using open data from Kamikatsu Town, Tokushima.

## Demo
https://code4fukui.github.io/become-money/

## Features
- 2-choice quiz game about whether items can become money
- Uses open data from Kamikatsu Town, Tokushima on recyclable items
- Displays the correct answer and recycling details after each question
- Shows the player's final score and accuracy percentage

## Data / API
The quiz data is sourced from the [Kamikatsu Town Recycling Open Data](https://github.com/code4fukui/zerowaste/blob/main/README.md) project.
